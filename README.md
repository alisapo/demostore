Проект представляет собой интернет-магазин обуви.

## Содержание

Приложение содержит следующие самостоятельные экраны (страницы):

+ Главная страница;
+ Каталог товаров;
+ Избранное;
+ Страница товара;
+ Оформление заказа.

И встроенные (являющиеся частью самостоятельных) экраны:
+ Товар;
+ Корзина покупок;
+ Поиск;
+ Подписка.
